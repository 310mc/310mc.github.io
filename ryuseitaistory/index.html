(Version 1.0) Ryuseitai Story List

Click HERE to go back to the homepage!
Welcome to the Ryuseitai story reading list! In here you'll find a list of Ryuseitai stories to get you started on reading about them (or to check what you haven't read so far). I'll start by explaining a few things.
  • Stories are picked based on lore and character relevance. For a complete list of every Ryuseitai appearance, please use the filters in the translation masterlist instead: EVENT/SCOUT TRANSLATIONS — IDOL/FEATURE TRANSLATIONS
    • Some stories have been temporarily omitted until there's an accessible translation.
    • Stories with a long list of character appearances have been shortened to the ones that interact with a Ryuseitai member.
  • This list is sorted in release order. For a chronological order of Ryuseitai's stories, please check the story timeline!
  • Stories are still being added and updated with information! Please feel free to contact me about the list or any errors on my ask box or on Twitter. Feedback is always welcome!
